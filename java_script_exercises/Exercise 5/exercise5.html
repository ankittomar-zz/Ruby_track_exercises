Exercise (Name input in Alert and Welcome in page content)

Create a HTML page, when it opens in Browser, it should prompt twice. Once for first name and second for last name. When you enter first and last name, it should alert you as "Hello firstname lastname."

After alert it should put that text in html page. Note: Keep in mind when you take inputs from users, always consider cases where user has entered null or blank values(few white spaces). In such cases you should not accept the input.
